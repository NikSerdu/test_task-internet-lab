# Test task Internet lab

## Installation

Using npm (port 5173)

```bash
  git clone https://github.com/NikSerdu/test_task-internet-lab
  cd test_task-internet-lab
  npm install
  npm run dev
```

Using docker compose (port 80)

```bash
  git clone https://github.com/NikSerdu/test_task-internet-lab
  cd test_task-internet-lab
  sudo docker compose up -d
```
